HTML Lists


HTML Lists allow web developers to group a set of related items
in lists

Ordered HTML list

An ordered list starts with the <ol> tag. Each list
item starts with the <li> tag.

Example:

<ol>
<li> Apple </li>
<li> Banana</li>
<li> Cherry</li>
<li> Date</li>
</ol>

Unordered HTML list

An unordered list starts with the <ul> tag. Each list
item starts with the <li> tag.

Example:

<ul>
<li> Coffee </li>
<li> Milk </li>
<li> Tea </li>
<li> Water </li>
</ul>

Output:

An ordered HTML list

  1. Apple
  2. Banana
  3. Cherry
  4. Date

An unordered HTML list

  • Coffee
  • Milk
  • Tea
  • Water
